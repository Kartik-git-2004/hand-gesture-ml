## 🚀 How It Works
- Detects hand gestures in **real-time** using OpenCV and cvzone.
- Tracks the distance between index fingers of both hands.
- Dynamically resizes and overlays an image based on the distance between fingers.

---

🛠️ Tech Stack
- 🐍 **Python**  
- 🎯 **OpenCV**  
- ✋ **cvzone**
